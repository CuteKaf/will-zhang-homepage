# FIR Lab Static Website

Pure static academic lab website for:

- `Frontier Intelligent Robotics Lab`
- `Yunbo "Will" Zhang`
- GitHub Pages deployment

## Location

`/Users/fanzhen/Documents/robotics/will-zhang-homepage`

## Preview

```bash
cd /Users/fanzhen/Documents/robotics/will-zhang-homepage
python3 -m http.server 4321
```

Open `http://localhost:4321`

## Pages

- `index.html`: homepage
- `research.html`: research directions
- `group.html`: professor and students
- `publication.html`: selected publications
- `news.html`: lab news
- `events.html`: lab events
- `positions.html`: join us
- `contact.html`: contact page

## Notes

- No framework
- No build step
- No JavaScript
- No animation
- Static HTML and CSS only
